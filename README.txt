# Patch v1-4-1 (UI/UX, 152-ФЗ, формы)

Содержит фиксы:
- Выровнен чекбокс и подпись на `/contacts`, отправка **только** после галочки и заполнения всех полей.
- `/ai-kp`: список шагов, сужен селект клиента, отступы, «Далее» не прилипает.
- `/ai-tok`: форма слева, результаты справа (две колонки).
- `/ai-doc`: описаны шаги и функциональность раздела.
- Layout/шрифты: `meta charset="utf-8"` и `@charset "UTF-8"` — защита от «кракозябр».

## Применение
Распаковать архив **в корень проекта** (где лежит папка `app/`), согласившись на замену файлов.

Либо через PowerShell из корня проекта:

```powershell
$patch = "ai-site-web_patch_v1-4-1"
Copy-Item -Path "$patch/app" -Destination "." -Recurse -Force
Copy-Item -Path "$patch/styles" -Destination "." -Recurse -Force
```

Затем:
```bash
pnpm dev
```
